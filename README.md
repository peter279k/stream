## Installation

```
composer require sunrise/stream
```

## How to use

```php
use Sunrise\Stream\Stream;

$handle = \fopen('http://php.net/', 'rb');

$stream = new Stream($handle);

var_dump($stream->toString());

$stream->close();
```
